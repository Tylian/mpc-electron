# Electron MPC client
A composable MPC client built upon Electron and Vue.js

## Running it
1. Install all dependencies:
    ```
    $ npm install
    $ npm install --global gulp
    ```

    or, using Yarn:
    ```
    $ yarn
    $ yarn global add gulp
    ```


2. Build and run it with gulp:
    ```
    $ gulp run
    ```
